# Demunger

Minimal Vite + React app with Tailwind CSS demonstrating DOMPurify.sanitize.

Getting started

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

Open the URL shown by Vite (usually http://localhost:5173).

What this app does

- Provides a textarea where you can paste HTML.
- Clicking the "Sanitize" button runs DOMPurify.sanitize on the input and displays the sanitized HTML below.
